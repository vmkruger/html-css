Principais Formatacoes

Negrito ou destaque

voce pode utilizar o negrito usando uma tag <b>nao semantica, apenas formato

voce tabem pode utilizar o negrito usando a tag <strong>, dando assim um sentido de semantica para o texto

Italico ou enfase

voce pode deixar um texto em italico usando a tag <i> (nao sematica)

voce tambem pode deixar um texto em italico usando a tag <EM> (semantica)

Texto Marcado

voce pode marcar um texto utilizando a tag <mark>

aqui esta outro exemplo de um texto marcado

Textos grandes e pequenos

Usando a tag <big>(tag obsoleta) podemos criar um texto grande e usando a tag <small>(tag ainda em uso) podemos criar um texto pequeno

Texto deletado

Podemos riscar/deletar um texto usando a tag <del>

Texto inserido

Podemos sublinhar/inserir um texto usando a tag <ins> ou usando a tag obsoleta <u>

Texto sobrescrito

Podemos escrever uma equacao numerica (x2+4x) ou a temperatura (200 celsius) utilizando a tag <sup>

Texto subscrito

Podemos escrever uma formula molecular (H20) usando a tag <sub>
